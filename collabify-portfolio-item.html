    What is Collabify?

    A web application that allows users to collaborate on Spotify queues real-time. Accessible at Collabify.

    Collabify is built using Python/Flask, Spotify API, and HTML/CSS/Boostrap

    What did I do?

    As the front-end engineer for the whole website, I created the initial layout of collabify and thought of the font and stylings of the mobile and desktop web verisons.

    I utilized Spotify's web api, Google fonts, and Flat icon to help customize the playback devices.

    What happened as a result?

    My team and I tested this project throughout several small gatherings throughout the school year.
